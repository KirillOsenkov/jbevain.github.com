Bio

Jb is an experienced software engineer with a passion for developer tools who took upon the challenge of creating a developer tools company: SyntaxTree.

He's the author of Cecil, a open-source .NET binary manipulation library that turned out to be useful to many people.

He's been involved in the Mono project (an open-source implementation of the Microsoft .NET framework) for over a decade. In his youth, he has been selected as a student in the very first two editions of the Google Summer of Code to contribute to Mono where he spent the summers flipping bits instead of burgers.

At SyntaxTree, Jb currently leads the development of UnityVS, a tool to bridge Microsoft Visual Studio and Unity, a powerful game creation editor. In just a short few years, UnityVS has growned to be used by thousands, from AAA studios to indies.

Prior to creating SyntaxTree, Jb achieved one of his dream when he joined the Mono team at Novell under Miguel de Icaza where he worked on various parts of the project. He was part of the team of 5 that started the MonoTouch and Mono for Android projects (later becoming the foundations of Xamarin), where he was in charge of the tooling to trim down the size of apps for deployment.

Before Novell, Jb worked at db4objects, a startup creating an open-source object oriented database for Java and .NET. There he worked not only on the database itself, written in Java, but also on the tooling to convert the Java code to .NET, and on .NET specific parts like the LINQ to db4o provider.
