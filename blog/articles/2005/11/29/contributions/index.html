Contributions 29 Nov 2005


Cecil have benefited from a few great contributions those last days:

Programmer's FAQ

Fabien Reinle, fellow friend from Evaluant have written a programmer's FAQ.

Generics support

Martin Baulig have completed the support of generics inside Cecil, so that it can be used in the debugger. It's a great news as Microsoft just released .net 2.

Regression test suite

Rodrigo just commited in /trunk/cecil/tests a regression test suite for Cecil. Make sure that you have NUnit, NAnt and boo installed, and press the red button!

Flow Analysis

Rodrigo, yes, the same, just commited in /trunk/cecil/flowanalysis his library I told you about in this blog entry. It can be used as a base layer for a lot of interesting applications. I'll blog on that later on.

Cecil 0.2 should be released in the next few days. Keep breathing!
