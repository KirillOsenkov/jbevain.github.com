Mono.Cecil 0.1 released 31 Oct 2005


Mono.Cecil 0.1 "Trick or Treat"

I'm very proud to announce the very first public release of the library Mono.Cecil.

Cecil is my first Open Source project of this size. It suffers from a few defaults, I learned a lot while working on it, but it's quite usable, and people seem happy to work with it.
